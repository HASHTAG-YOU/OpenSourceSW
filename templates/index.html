4. 새 키 페어 생성 후 다운로드

5. puttygen에서 4에서 다운로드 한 키 파일(.pem) load 후 Save private key 클릭하여 저장

6. putty 왼쪽 Category - Connection - SSH - Auth private key file for authentication 에서 browse 클릭하여 5에서 저장한 .ppk 파일 선택

7. putty 왼쪽 Category - Session 으로 가 Host name 입력 후, Open 클릭
